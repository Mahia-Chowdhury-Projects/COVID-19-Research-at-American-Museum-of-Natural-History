Visualizations

Choropleth Maps

Covid-19 Positivity Rates Overtime

This choropleth map displays Covid-19 positivity rate worldwide. Positivity rates are the percentage of all coronavirus tests performed that are actually positive. Public health officials rely on the results to track the state of the pandemic, and policymakers use this information to guide decisions on reopening schools and businesses.

Number of Covid-19 Deaths

This map displays the number of deaths due to Covid-19 in all U.S States based on a date users input. As the colors get darker the deaths increase.

Case-Fatality Rates

This Map displays fatality rates for U.S states based on date a users input. Case fatalities are the proportion of deaths from a certain disease compared to the total number of people diagnosed with the disease.

Prediction Model

Fully Vaccinated Model

This graph predicts the number of fully vaccinated individuals based on the days given to the prediction function, in this case 90 days. This information can be used to predict when restriction will be lifted and deduce when it is safe to go outside
